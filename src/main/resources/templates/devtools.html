devtools

IntelliJ, SpringBoot devtools livereload

  1. Add LiveReload extension into your browser. chrome

  2. Add devtools dependencies to your "pom.xml". Add the following code in your dependencies section for maven .

    
        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-devtools</artifactId>
            <scope>runtime</scope>
        </dependency>

  3. In your intellij IDEA go to: file->settings->build,execution,deployment. Go to ->compiler->build project automatically.

  4. In your intellij IDEA: Ctrl + Shift + A.
    type "registry"
    click "compiler.automake.allow.when.app.running"

  5. Press F12 or
    (Ctrl + Shift + I).
    Go to Network and Click on "Diable Cache" which will not save the page into cache.
    When page refreshes it shows changes
